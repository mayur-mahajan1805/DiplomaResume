# **App Name**: CareerCompass AI

## Core Features:

- Resume Parsing & Skill Extraction: AI tool parses the uploaded resume (PDF or text), extracts relevant information such as technical skills, soft skills, work experience, and education using LLM reasoning.
- Job Role Recommendation: Recommends top 3-5 job roles based on skills extracted from the resume, providing a match percentage for each role, and reasoning from the AI.
- Skill Gap Identification: Identifies missing skills required for the recommended job roles, offering suggestions for learning resources and potential career development paths using AI.
- Interactive Career Chat: Provides a chatbot interface where users can ask career-related questions and receive tailored advice powered by generative AI. The AI reasons over a knowledge base of career advice.
- User Profile and Resume Storage: Allows users to create profiles, upload, and store their resumes securely. Each resume is parsed and the data is stored, enabling easy access and management.
- Resume Upload and Display: Enables users to upload resumes in PDF or text format and displays the extracted data in a structured and readable manner, confirming the parsing quality.
- Clear Data Visualization: Presents skills, job recommendations, and skill gaps through visually intuitive charts and graphs, improving user understanding and decision-making.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke feelings of professionalism, intellect, and trustworthiness.
- Background color: Very light grayish-blue (#F0F8FF).
- Accent color: Teal (#008080), chosen as an analogous hue offering contrast through both brightness and saturation, while complementing the primary's serious tone with an impression of innovation.
- Body and headline font: 'Inter' sans-serif for a modern and neutral look that is suitable for both headlines and body text.
- Use a set of consistent and modern icons, specifically Feather icons or Font Awesome, to visually represent different skills, job roles, and career advice categories.
- Implement a clean and responsive grid-based layout that adapts seamlessly to different screen sizes, ensuring readability and ease of navigation across devices.
- Incorporate subtle transitions and animations for loading states and data updates to enhance the user experience without being distracting.
